# 🚀 Crypto Tracker Dashboard

## 📌 Project Overview

The **Crypto Tracker Dashboard** is a frontend web application that displays real-time cryptocurrency data using a public API.
It allows users to explore different cryptocurrencies, view their latest prices, and interact with the data through search, filtering, and sorting features.

This project is designed to demonstrate core concepts of JavaScript, API integration, and UI development**.

---

## 🎯 Objectives

* Fetch and display real-time data from a public API
* Implement dynamic UI updates using JavaScript
* Use array higher-order functions (HOFs) like `map`, `filter`, `sort`, and `find`
* Build a responsive and user-friendly interface

---

## 🌐 API Used

* **CoinGecko API**
* Endpoint:

  ```
  https://api.coingecko.com/api/v3/coins/markets?vs_currency=usd
  ```

---

## ✨ Features

### 🔍 Search

* Search cryptocurrencies by name or symbol

### 🎛️ Filtering

* Filter coins based on performance (e.g., gainers or losers)

### 🔽 Sorting

* Sort cryptocurrencies by:

  * Price (ascending/descending)
  * Market Cap

### ❤️ Favorites

* Mark coins as favorites
* Store favorites using **localStorage**

### 🌙 Dark Mode

* Toggle between light and dark themes

### ⏳ Loading Indicator

* Displays a loading state while fetching data
